Designed to run smoothly on laptops with 8GB RAM.🚀 Key FeaturesModel: Llama 3.2 3B (Lightweight & Smart).Privacy: 100% Local (Ollama + FAISS).Features: Chat, File Upload (PDF/DOCX), Analytics Dashboard, Feedback System (CSAT).🛠 Quick Start1. Install & Prepare OllamaCoreMind now uses llama3.2:3b for better performance on 8GB RAM.Download Ollama from ollama.com.Open PowerShell and run:ollama pull llama3.2:3b
(Wait for it to download ~2.0 GB).Ensure Ollama is running and listening on 0.0.0.0 (for Docker support):$env:OLLAMA_HOST = "0.0.0.0"
ollama serve
2. Run with DockerIn your project folder:docker-compose up --build
3. AccessUI: http://localhost:8501API: http://localhost:5000📂 Project Structureapi.py: Backend logic (Flask + RAG).app.py: Frontend UI (Streamlit).memory_utils.py: Vector database management (FAISS).data/: Folder where your uploaded files and database live.
